Metaetiquetas en html

¿Qué son las metaetiquetas?

Las metaetiquetas son fragmentos de código HTML que proporcionan informacion sobre una página web. No son visibles en la página pero los motores de búsqueda las utilizan para diferentes propósitos.

Ejemplo de metaetiquetas comunes

  • Meta de descripción: Proporciona un resumen de la página. Ejemplo: 
    < meta name= "description" content= "Sitio de noticias">
  • Meta de palabras claves: Lista de térnminos relacionados con el contenido. Ejemplo: 
    <meta name= "keywords" content= "deportes, noticias, clima, farándula, finanzas, política" >
  • Meta de author Indica el nombre del creador del sitio. Ejemplo: 
    <meta name= "author" content= "Olivia Galaviz">
